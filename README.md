# ProtractorSandbox
## Running the tests

To run the tests, in a command window, type:
<pre>protractor conf.js</pre>

This assumes that
<ol>
  <li> you have a file conf.js in the current directory, </li>
  <li> for anything to happen, that conf.js references a spec2.js (right now) </li>
</ol>
